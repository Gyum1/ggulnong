이달의 식물

완두콩

1. 파종깊이 5cm, 파종간격 15cm~20cm정도로 심는다.
2. 한 구멍에 완두콩 씨앗 2~3알씩 파종한다.
3. 파종 후 10일~15일이면 싹이 나오고,
35일~40일 사이에 꽃이 핀다.
4. 꽃이 핀 후 10일 정도 지나면 꼬투리가 열린다.
5. 파종 후 보통 80일~90일 경에 수확한다.
